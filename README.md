# 🌟 Modern Login & Signup Page

A **sleek, interactive, and responsive** Login & Signup page built using **HTML, CSS, and JavaScript**. Designed with modern UX/UI features for a smooth and engaging user experience.  

## ✨ Features

- 🔄 **Sliding Transition** – Smooth switch between **Sign In** and **Sign Up** forms  
- ✅ **Remember Me** – Keep users logged in across sessions  
- 👁️ **Password Eye Icon** – Toggle password visibility  
- 🌙 **Dark Mode** – Switch between light & dark themes  
- 🔢 **Character Count** – Real-time feedback on input fields  
- ⚠️ **Error Handling** – Validations for empty fields, invalid emails, weak passwords, and mismatched passwords  
- 🔗 **Social Login Buttons** – UI ready for **Google, GitHub, and Facebook** integration  

## 🛠️ Technologies Used

- **HTML5** – Page structure  
- **CSS3** – Styling, animations, sliding effects, dark mode  
- **JavaScript** – Form validation, sliding forms, password toggle, character count  

## ⚡ How It Works

- **Sliding Forms** – CSS animations + JS toggles for smooth switching  
- **Password Eye Icon** – Toggle input type between `password` and `text`  
- **Remember Me** – Simulated using `localStorage` (frontend only)  
- **Character Count** – JS updates character count in real-time  
- **Error Handling** – JS prevents invalid submissions and shows user-friendly messages  
- **OAuth Buttons** – Styled for **Google, GitHub, Facebook** (backend integration optional)
